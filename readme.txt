Requirements:
Python 3 (https://www.python.org/downloads/release/python-385/)
streamlit (https://docs.streamlit.io/en/stable/)
pandas (pip install pandas; or go to https://pandas.pydata.org/pandas-docs/stable/getting_started/install.html)
numpy (pip install numpy; or go to https://phoenixnap.com/kb/install-numpy)
scikit-learn (pip install sklearn; or go to https://scikit-learn.org/stable/install.html)

Run the script from terminal with 'streamlit run data_analysis.py'
